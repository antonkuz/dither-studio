## Project Overview

A vanilla JS browser demo for ordered (Bayer 4x4) dithering of images and video. No build system, no dependencies to install — open `index.html` directly in a browser.

## Running

```sh
open index.html
# or serve with any static file server:
python3 -m http.server
```
